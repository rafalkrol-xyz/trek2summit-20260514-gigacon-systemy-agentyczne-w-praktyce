## 🛠️ Jak uruchomić prezentację?

Prezentacja jest napisana w [Markdownie](https://en.wikipedia.org/wiki/Markdown)
i uruchamiana poprzez [Presenterm](https://github.com/mfontanini/presenterm).

```bash
# Instalacja (macOS)
brew install presenterm

# Uruchomienie
presenterm presentation.md
```
